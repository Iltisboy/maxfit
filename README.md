# MaxFit – Dein Fitnesstracker

Deine persoenliche Trainings-App.

## Deployment auf Vercel (einmalig, 10 Minuten)

### Schritt 1: GitHub-Account erstellen
1. Gehe zu https://github.com
2. Klicke "Sign up"
3. E-Mail, Passwort, Username eingeben - fertig

### Schritt 2: Repository erstellen
1. Auf GitHub: Klicke oben rechts "+" - "New repository"
2. Name: `maxfit`
3. Klicke "Create repository"
4. Lade die Dateien hoch: "uploading an existing file" - alle Dateien aus diesem Ordner reinziehen - "Commit changes"

### Schritt 3: Vercel verbinden
1. Gehe zu https://vercel.com
2. Klicke "Sign up" - "Continue with GitHub"
3. Klicke "Add New..." - "Project"
4. Waehle dein `maxfit` Repository
5. Framework: "Vite" (wird automatisch erkannt)
6. Klicke "Deploy"
7. Warte 1-2 Minuten - deine App ist live!

### Schritt 4: Auf iPhone installieren
1. Oeffne Safari auf dem iPhone
2. Gehe zu deiner URL (z.B. `maxfit.vercel.app`)
3. Tippe auf das Teilen-Symbol (Quadrat mit Pfeil)
4. Scrolle runter - "Zum Home-Bildschirm"
5. Tippe "Hinzufuegen"
6. Fertig! MaxFit ist jetzt auf deinem Home-Bildschirm.

## Lokale Entwicklung (optional)

```bash
npm install
npm run dev
```
